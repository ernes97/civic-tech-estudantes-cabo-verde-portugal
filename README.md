# 📊 Civic Tech: Estudantes de Cabo Verde no Ensino Superior Português

**Análise de dados oficiais DGES (2015-2020) | Python + Plotly**

### O que o projeto responde
- Qual foi o crescimento real dos estudantes cabo-verdianos em Portugal?
- Como Cabo Verde se posiciona face aos outros países PALOP?
- Qual o impacto da CPLP na mobilidade académica?

### 🔑 Principais Insights

- **Cabo Verde registou um CAGR de +47.8%** entre 2015-2019 (Compound Annual Growth Rate), um dos crescimentos mais fortes entre os países PALOP.
- Em 2019 representava **38.7%** do total de estudantes africanos PALOP no ensino superior português.
- Tendência clara de aceleração após 2017, com o número de estudantes quase a duplicar entre 2017 e 2019.
